CREACIÓN DE TIENDAS VIRTUALES

           

            Este capítulo describe la creación de tiendas virtuales con el manager de configuración del programa, también se verá la creación y administración de proyectos y su compilación.

 

Arranque del manager de Windows de Marrok Shoping Cart.

      

       Para arrancar el programa puede seguir uno de los dos siguientes caminos:

 

1-        En el escritorio pulsar dos veces el icono Marrok Shoping Cart.

 

2-        Pulsar en el menú inicio - todos los programas - Marrok Shoping Cart - Marrok Shoping Cart.

 

       En ambos casos tras un breve lapso de tiempo aparecera una pantalla como la siguiente, desde la que podrá elegir crear un nuevo proyecto, abrir uno existente o cerrar dicha ventana.

 

 

 

Gestión de proyectos

           

            Las tiendas que creé con el programa se almacenan bajo una serie de proyectos en el programa, con estos proyectos se puede hacer las siguientes operaciones:

 

·       Crear un nuevo proyecto: para ello puede seleccionar el botón la pantalla anterior, o la opción correspondiente del menú Archivo o de la barra de herramientas del programa; con lo que aparecerá una ventana como la siguiente.

 

.

 

En la que podra escribir un nombre de proyecto válido (solo ha de contener lentras, excluyendo la ñ, y números y empezar con una letra) y pulsar Aceptar para crearlo.

                       

 

·       Abrir un proyecto: para ello puede seleccionar el botón la pantalla anterior, o la opción correspondiente del menú Archivo o de la barra de herramientas del programa; con lo que aparecerá una ventana como la siguiente.

 

 

                       

En dicha ventana puede abrir cualquiera de los proyectos creados anteriormente, por defecto, hay uno llamado DEMOTIENDA, el cual es un ejemplo para ver las capacidades del programa.

 

·       Borrar un proyecto: para lo cual puede usar la opción del menú Archivo o de la barra de herramientas, con lo cual aparecerá una ventana como la de abrir proyectos; pero en este caso se usará para borrarlos, sin embargo, hay que tener encuentra que no se puede borrar un proyecto abierto.

 

 

 

Configuración de tiendas virtuales

 

Una vez creado o abierto un proyecto aparecera una ventana como la siguiente:

 

 

 

La cual esta dividida en catorce pestañas para configurar la tienda, a continuación se explica cada una de ellas:

 

·        General:

 

1- Opciones de la página: permite especificar el título y el modelo de presentación de la página.

 

2- Opciones de lenguaje: sirve para especificar los lenguajes que se usaran en la página, y el lenguaje de presentación por defecto.

 

3- Opciones de las monedas: permite especificar la moneda principal que se usara, y si se desea poner el precio en una moneda secundara y su nombre.

      

·         Marco 1: si el modelo de presentación es el 2,3,4 o 5 estas opciones configuran el marco adyacente, pero si es el modelo 6,7,8 o 9 estas opciones configuran el marco vertical.

 

       1- Opciones a incluir: permiten elegir las opciones de enlace que aparecerán en el marco, estas opciones son:

 

A- Incluir barra de navegación: para añadir los enlaces que te permiten navegar por las distintas secciones de la tienda.

 

B-Incluir buscador: permite poner un enlace para desplegar las opciones de búsqueda.

 

C-Incluir lenguaje: permite añadir un enlace para poder activar la elección de lenguaje de la tienda.

 

D-Incluir disclaimer: añade un enlace para desplegar el disclaimer.

           

2- Fuentes del marco 1: permite elegir el tipo, tamaño y color de la fuente que se usara por defecto en el marco 1.

           

*Nota: aunque casi siempre se permite elegir estos tres atributos de las fuentes del programa a veces, no tendrán efecto, en algunos casos concretos el valor que se aplique al tamaño y al color.

                       

3- Opciones de los enlaces: deja elegir el color que tendrá los enlaces visitados y sin visitar.

           

4- Misceláneo: Esta compuesta por las siguientes opciones:

 

A- Imagen fondo: permite especificar la imagén de fondo del marco.

 

B- URL del marco: si no se activan ninguna de las opciones del grupo "Opciones a incluir" se puede sustituir documento por defecto del mismo por otro creado por el usuario.

 

C- Ancho del marco: permite especificar el porcentaje de ventana que ocupa el marco.

 

D- Color de fondo del marco: permite especificar el color de fondo por defecto del marco.

 

·         Marco 2: si el modelo de presentación es el 6,7,8 o 9 estas opciones configuran el marco horizontal.

 

       1- Opciones a incluir: permiten elegir las opciones de enlace que aparecerán en el marco, estas opciones son:

           

A- Incluir barra de navegación: para añadir los enlaces que te permiten navegar por las distintas secciones de la tienda.

 

B- Incluir buscador: permite poner un enlace para desplegar las opciones de busqueda.

 

C- Incluir lenguaje: permite añadir un enlace para poder activar la elección de lenguaje de la tienda.

 

D- Incluir disclaimer: añade un enlace para desplegar el disclaimer.

 

            2- Fuentes del marco 2: permite elegir el tipo, tamaño y color de la fuente que se usara por defecto en el marco 2.

 

            3- Opciones de los enlaces: deja elegir el color que tendrá los enlaces visitados y sin visitar.

 

            4- Misceláneo: Esta compuesta por las siguientes opciones.

                                   

A- Imagen fondo: permite especificar la imagen de fondo del marco.

 

B- URL del marco: si no se activan ninguna de las opciones del grupo "Opciones a incluir" se puede sustituir documento por defecto del mismo por otro creado por el usuario

    

C- Ancho del marco: permite especificar el porcentaje de ventana que ocupa el marco.

    

D- Color de fondo del marco: permite especificar el color de fondo por defecto del marco.

 

·     Marco central: Hace referencia a la sección principal de la pagina, donde se ve la presentación de la tienda, los productos etc.

 

       1- Opciones a incluir: permiten elegir las opciones de enlace que aparecerán en el marco, estas opciones son:

           

A- Incluir barra de navegación: para añadir los enlaces que te permiten navegar por las distintas secciones de la tienda.

    

B- Incluir buscador: permite poner un enlace para desplegar las opciones de búsqueda.

 

C- Incluir lenguaje: permite añadir un enlace para poder activar la elección de lenguaje de la tienda.

    

D- Incluir disclaimer: añade un enlace para desplegar el disclaimer.

           

2- Fuentes del marco central: permite elegir el tipo, tamaño y color de la fuente que se usara por defecto en el marco central.

 

            3- Opciones de los enlaces: deja elegir el color que tendrá los enlaces visitados y sin visitar, y si se situaran los enlaces de control arriba de la página, abajo en  ambos  sitios.

           

4- Misceláneo: Esta compuesta por las siguientes opciones.

           

A- Imagen fondo: permite especificar la imagen de fondo del marco.

    

B- Color de fondo del marco: permite especificar el color de fondo por defecto del marco.

 

·     Buscador: Engloba las opciones para configurar el interfaz de búsqueda de la tienda.

 

            1-Colores del buscador: configura los colores que se usaran en la capa del buscador, en la tabla que contiene el interfaz de búsqueda y el borde de la misma.

 

            2-Campos de búsqueda: Establece los datos por los que se pueden buscar los productos.

 

            3-Fuentes del buscador: Establece la fuente de texto del buscador.

      

·     Categorías: Permite modificar las distintas opciones de presentación de la página donde se muestran las categorías.

 

       1- Posición de la tabla de categorías: permite fijar el ancho de la tabla donde se muestran las categorías y el sangrado de la misma donde se empezara a escribir el árbol de subcategorías.

           

2- Fuente de las categorías: Permite establecer la fuente con la que se mostraran las categorías.

           

3- Fuente de las subcategorías: Permite establecer la fuente con la que se mostraran las subcategorías.

 

4-Número de categorías por página: Establece el número máximo de categorías que se mostraran por página, si hay mas categorías aparecerán enlaces para navegar por las distintas páginas.

 

5-Color de fondo de las categorías: Establece el color para las celdas pares e impares de la tabla donde se mostrarán las categorías

 

·     Minifichas: Establece la apariencia visual de las fichas resultantes de una búsqueda por los productos disponibles, o de elegir una subcategoría de la página categorías.

 

1- Tamaño de las minifichas: Establece el ancho de la tabla donde se muestran las minifichas.

 

2- Color fondo de las minifichas: Establece el color para las celdas pares e impares de la tabla donde se mostrarán las minifichas.

 

3- Tamaño de las imágenes de las minifichas: Establece el tamaño en píxeles de la imagen de la minificha.

 

4- Fuentes de las minifichas: Establece la fuente que se usará en las minifichas.

 

5- Número de productos por página: Establece el número de minifichas que se mostrarán por página.

 

·     Fichas: Establece la apariencia visual de la ficha ampliada del producto que se desea consultar.

 

1- Fuente de las fichas: Establece la fuente que se usara en las fichas.

 

2- Colores de fondo de las fichas: Establece el color de fondo de la ficha y de las casillas que contienen los campos de la ficha.

 

3- Tamaño de las imágenes de las fichas: Establece el tamaño de la imagén de las fichas en pixels.

      

·     Ver Carro: Permite modificar la apariencia de la página donde se listan los artículos añadidos al carro.

 

1- Color de la cesta: Permite establecer el color de fondo de la tabla y de las casillas de los campos de la página donde se listan los artículos añadidos al carro

 

2- Fuentes de la cesta: Establece la fuente que se usara en la página donde se listan los productos añadidos al carro.

 

·     Página de comprar carro: Establece la apariencia y las opciones del formulario de compra de productos.

 

1- Fuentes de la página: Establece la fuente que se usara en la página donde se solicita la compra de los productos del carro.

 

2- Colores de fondo del formulario del pedido: Establece el color de fondo de la tabla del formulario donde se realizara la solicitud del pedido y las celdas donde están los campos de dicho formulario.

 

3- Opciones de la licencia: Establece si desea incluir una licencia que deba aceptarse en el formulario de pedido, y si desea que se muestre en una ventana aparte o en una capa dentro de la página.

 

4- Opciones de la compra: Permite añadir opciones adicionales al formulario donde se realiza el pedido.

 

A- Dejar que el usuario pueda guardar su pedido: Añade una casilla de selección al formulario para que el usuario pueda, si lo desea, guardar su pedido de sesión en sesión con un cookie. También puede especificar el número de días de vida del cookie con la opción Días de vida del cookie.

                            

B- Permitir que se escriba un comentario: Añade una caja de texto para que el usuario pueda escribir un comentario u opinión sobre el servicio que proporciona la tienda.

 

C- Permitir que se especifiquen instrucciones especiales :Añade una caja de texto para que el usuario pueda escribir un instrucciones especiales para el envío del pedido.

 

5- Pagos: Especifica las distintas opciones de pago permitidas en la tienda.

                            

A- Paypal: Permite usar dicho servicio para el pago electrónico del pedido, además tendrás que indicar el correo de electrónico de paypal en la casilla Correo-e de paypal.

 

B- Transferencia bancaria: para permitir el pago solicitando que el usuario envié el reembolso del pedido a un número de cuenta bancaria, la cual, se especificará en la casilla Cuenta bancaria.

 

C- Contra reembolso: para permitir al usuario el pago del pedido cuando le llegue a su domicilio.

 

·     Página final: Permite modificar la apariencia visual de la página final donde están las instrucciones de pago del pedido.

 

1-Fuentes de la página: Especifica la fuente que se usará en los textos de dicha página.

                       

2-Color de fondo de la página final: Especifica los colores de las tabla exterior e interior donde esta las instrucciones de pago del pedido.

 

·     SMTP y contraseñas: En esta pestaña se puede especificar la contraseña para encriptar el pedido y los datos de la cuenta de correo donde se enviara el pedido.

 

1- Contraseña de encriptación: Establece la contraseña con la que se encriptará el pedido y otra información sensible de la tienda.

 

2- Opciones del correo electrónico de recepción de pedidos: En esta parte se puede especificar la dirección de correo electrónico donde se te enviara el pedido al igual que la dirección y puerto del servidor SMTP el asunto del mensaje, también permite especificar el login y password del servidor

 

·     Textos, disclaimers y licencias: Un conjunto de opciones variadas relacionadas con los disclaimer, licencias textos de la página y la factura.

 

1- Fuentes del disclaimer: Especifica la fuente que se usara en el disclaimer o la licencia.

 

2- Fuente para indicar que no hay resultados: Especifica la fuente que se usara para mostrar el mensaje oportuno cuando no haya resultados en la búsqueda realizada con el buscador de la tienda.

 

3- Datos de la factura: Permite especificar datos adicionales a la factura como los datos de la localización de la empresa, el cif o una imagen.

                            

A- Datos empresa: Permite especificar texto adicional a para la factura, también están permitidas las etiquetas HTML.

 

B- Logotipo empresa: Permite especificar un archivo de imagen que se usará en la parte superior de la factura.

                       

4-  Opciones varias: Permite especificar si el disclaimer se mostrara en una ventana aparte o en una capa html.

 

5- Opciones de edición de texto: permite editar el texto de la página principal, el disclaimer y la licencia, este tema será tratado más adelante.

 

·     Productos: Permite añadir, modificar y borrar productos, esta pestaña sera detallada mas adelante.

 

·     Edición de los textos de la página principal, disclaimers y licencias.

 

Para editar dichos textos hay que seleccionar la pestaña Textos, licencias y disclaimers, en dicha pestaña en la sección opciones de edición de textos aparecerá una lista con los lenguajes añadidos, seleccionado uno de los lenguajes y pulsando uno de los botones de la derecha aparecerá un editor de textos como el siguiente:

 

 

       En dicho editor se podrá editar el texto de dichos apartados, el editor se parece al bloc de notas de Windows. También en el editor, si se esta creando la página principal de la tienda, podrás previsualizar el como va quedando la página y añadir etiquetas html.

 

            Como mínimo con el editor tendrá que crear una página inicial para tienda por cada lenguaje añadido, además de la licencia y el disclaimer si se ha activado dichas opciones.

 

·     Añadir modificar y borrar productos.

 

       Para hacer esto hay que dirigirse a la pestaña productos, en dicha pestaña, que en un principio aparecerá vacía, se podrá añadir productos pulsando el botón añadir o modificar un producto seleccionando uno de la lista y pulsando el botón añadir, con lo cual aparecerá una ventana como esta.

 

 

       Esta ventana esta dividida en tres pestañas que detallamos a continuación.

 

1- Producto: la cual esta compuesta por los siguientes campos:

 

A- Nombre del producto: campo obligatorio que especifica el nombre del producto, si se esta modificando el producto este campo no se puede modificar.

 

B- ID del producto: esta campo, que también es obligatorio, es un identificador interno del producto.

                

C- Categoría: especifica la categoría donde se engloba el producto este campo es obligatorio.

 

D- Subcategoría: especifica la subcategoría donde se engloba el producto este campo es obligatorio.

E- Descripción de producto: una descripción opcional del mismo.

 

2- Modelo: sección donde se añade eliminan y modifican los modelos del producto, esta sección tiene los siguientes campos.

 

A- Modelo: es un campo obligatorio el cual especifica el nombre del modelo

 

B- Precio1: el precio de modelo en la moneda primaria, es un campo obligatorio

 

C- Precio2: es un campo obligatorio, el cual especifica el precio en la moneda secundaria de la tienda

 

D- Descuento: especifica el porcentaje de descuento de dicho modelo, es opcional.

 

E- IVA: especifica el porcentaje de impuesto de dicho modelo, es opcional.

 

3- Imagen: permite añadir una imagen del producto esta compuesta por los siguientes campos.

 

A- Nombre de imagen: especifica un nombre simbólico a la imagen, el cual es obligatorio.

 

B- ID de la imagen: un identificador interno de la imagen, el cual es obligatorio.

 

C- Copyright: permite añadir los derechos de autor de la imagen en la página.

 

D- Texto alternativo: permite añadir un texto alternativo que se visualizara si la imagen no se puede mostrar.

 

E- Descripción: permite añadir una descripción a la imagen útil por si la imagen no se muestra o para navegadores de invidentes.

 

G- Archivo: permite seleccionar el archivo de imagen que se usara, para ello puede usar el botón examinar para recorrer el árbol de directorios del disco      duro.

 

·     Compilar el proyecto

 

Para compilar el proyecto hay que acceder a la opción proyecto - compilar de menú inicio o pulsar el botón correspondiente de la barra de opciones, con lo cual, aparecerá una ventana en la que se te dirá que antes hay que guardar el proyecto, pulse sobre si para guardar el proyecto. Una vez guardado se verificara que está completa toda, si no es así ira mostrando una serie de alertas donde se indica lo que falta por completar.

 

Si no falta nada al cabo de unos segundos aparecerá una ventana donde se te preguntara donde guardar la tienda, navegue por el árbol de directorios hasta encontrar el directorio donde desea guardarlo y puse aceptar, al cabo de un momento aparecerá una pantalla de confirmación.

 

·     Previsualizar el proyecto

 

Para previsualizar el proyecto hay que acceder a la opción proyecto - previsualizar de menú inicio o pulsar el botón correspondiente de la barra de opciones, con lo cual, aparecerá una ventana en la que se te dirá que antes hay que guardar el proyecto, pulse sobre si para guardar el proyecto. Una vez guardado se verificara que está completa toda, si no es así ira mostrando una serie de alertas donde se indica lo que falta por completar.

 

Si no falta nada al cabo de unos segundos aparecerá una ventana como la siguiente:

 

                       

En la cual se te pedirá la ip y el puerto donde se pondrá a escuchar el servidor web, esta ventana solo aparecerá una vez, a no ser que abras otro proyecto o cierres el programa. Al cabo de un rato se abrirá el navegador donde se previsualizara la tienda.
