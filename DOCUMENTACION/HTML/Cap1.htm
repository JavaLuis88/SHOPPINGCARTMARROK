Instalación y desinstalación de Marrok Shopping Cart

 

       Este capítulo trata sobre la instalación paso a paso del programa en Windows y su desinstalación, en el proceso de instalación se copian los archivos necesario, se registra la aplicación en el sistema para su posterior desinstalación y se crean los accesos directos del mismo.

 

·       2.1 Instalación del programa

 

Para instalar Marrok shoping Cart, se han de seguir los siguientes pasos:

 

1-  Descargue el archivo comprimido del programa en un directorio temporal.

 

2-  Descomprima el archivo, lo cual generara una serie de archivos.

 

3-  Ejecute el archivo setup.exe el cual, procederá ,si es necesario, instalando el windows installer y a mostrar el programa que le guiara en la instalación.

 

4-  En la primera pantalla de bienvenida, se explica que he es necesario para que se instale correctamente el programa, que se cierre cualquier aplicación abierta. Si este es el caso, puede pulsar cancel el cualquier momento para cancelar la instalación y realizar este paso, o si no desea instalar la aplicación en este momento.

 

5-  Pulse sobre next.

 

6-  En la siguiente pantalla aparecerá la licencia, la  cual, ha de ser aceptada para poder instalar el programa, por eso, tómese su tiempo para leerla, si la acepta pulse sobre i agree y next para continuar.

 

7-  A continuación se le pedirá que elija el directorio donde se instalara la aplicación, por defecto aparece uno pero, puede cambiarlo escribiendo otro distinto a pulsando sobre el botón browse. Una vez elegido el directorio pulse sobre next.

 

8-  En la ultima pantalla antes del proceso de copia de los archivo, se le indica que el programa ya tiene los datos necesarios para la instalación, aquí puede cambiar alguno de los datos de pantallas anteriores pulsando sobre previous, o salir de la instalación pulsando cancel. Pulse next para continuar la instalación

 

9-  Los archivos se copiaran donde hayas decidido instalar el programa, una vez copiados aparecerá otra pantalla indicando el final del proceso, pulse sobre finish para terminar la instalación, es posible que deba reiniciar el ordenado si este es el caso proceda así con el consiguiente arranque del ordenador.

 

Una vez instalado se creara un acceso en el escritorio al programa Marrok Shoping Cart, y un grupo en el menú de inicio  todos los programas llamado también Marrok Shoping Cart con los siguientes accesos directos: Asistente de desencriptación de mensajes, Licencia de Marrok Shoping Cart, Manual de Marrok Shoping Cart, Marrok Shoping Cart, Registrar Marrok Shoping Cart, Seleccionar Lenguaje.

 

·       2.2 Desinstalación del programa

 

Para desinstalar el programa, puede usar las capacidades de desinstalar de Windows, para ello seleccione Inicio  Panel de control  Agregar quitar programas, seleccione la entrada Marrok Shoping Cart pulse quitar y siga los pasos que se den para la desinstalar el programa.
