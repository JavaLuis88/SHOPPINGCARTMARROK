·       ¿Qué es Marrok Shoping Cart?

 

       Es un programa para la generación de tiendas virtuales para internet fácil de utilizar y manejar incluso por personas sin conocimientos en el desarrollo de aplicaciones web, además de tener una gran cantidad de opciones para crear la tienda lo mas personalizada posible, las principales características de la

misma son:

 

-        Nueve maneras de presentación de la tienda.

-        Pleno control en la elección de fuentes y colores.

-        Pagos electrónicos con paypal.

-        Envío del pedido del comprador al correo del vendedor cifrado.

-        Precios en dos monedas.

-        Fácil inclusión de licencias y disclaimers.

-        Posibilidad de presentar la tienda en varios lenguajes.

-        No requiere de ningún programa en el servidor, ya que todo se realiza en el cliente.

-        Instalación automática de la maquina virtual de java cuando es necesario.

-        Manager de Windows para una fácil configuración de las tiendas, dicho programa también es multilenguaje.

-        Muchas más cosas incluidas.

 

·       Requerimientos para la instalación del programa.

 

Para instalar este programa es necesario unos requerimientos mínimos estos son:

 

       - PC con Windows nt, 2000, xp u otro windows basado en tecnología nt.

-        128 megas de ram.

-        40 megas de disco duro

-        Raton y teclado

 

Por su parte  los visitantes de las tiendas creadas requieren un navegador con:

 

-        Javascript 1.2

-        Java

-        Cookies

-        Capas W3C
